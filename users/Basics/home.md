# Welcome to CodeNForce
One of the many goals of the Turtle Creek Valley Council of Governments (COG) involves working towards mitigating blight in its member and neighbor communities. By implementing an effective code enforcement program available to code officers and borough staff alike, blighted property conditions can be addressed in an effective, timely manner. Developed and housed at TCVCOG, CodeNforce is built with support from the State of Pennsylvania and the Hillman Foundation to meet the real-world needs of municipal staff and inspectors. The system was designed as an affordable solution that struggling towns can use to modernize their operations—streamlining property maintenance code enforcement and occupancy inspections. CodeNForce now does all that and more for twelve communities across Allegheny County and Westmoreland County.

CodeNForce was built and is updated with the input of property maintenance inspectors in order for the tools to work best for them. What sets CodeNforce software apart from others is its ability to share data across municipal boundaries to leverage Act 90 and hold problem property owners accountable. Every parcel, owner, and local ordinance is built directly into the system, capturing all the important details a code enforcement officer handles throughout the day. Coupled with various tools and features like document generation, citation tracking, permitting, and event reminders all built into the system, CodeNforce empowers all municipalities to collaborate in addressing blight, monitoring compliance, and protecting residents. CodeNForce is a project that will continue to improve as more towns join to share their data. Protecting our communities from seeing the economic damage caused by blighted conditions can only happen as towns work together to form robust, efficient code enforcement programs. CodeNForce, with all its tools and aids, can make that happen in every borough that joins the cause. 

Throughout the site, users can find guides on how to use the various functions of CodeNForce to aid in streamlining and digitizing their code enforcement programs, processes, and requirements. The more data added into the municipailty's database, the better the program works. If any further issues present, please reach out to the Turtle Creek Valley Council of Governments for technichal assistance.
