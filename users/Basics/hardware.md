# CodeNforce hardware and software specs
The following hardware and software specifications ensure compatibility with our software base and a minimally funational user experience. 

## Computer requirements for CNF browser application
The primary CodeNforce tool is served to users via a standard web browser which means we are operating systems agnostic. The CNF officially supported browser is [Google Chrome](https://www.google.com/chrome/index.html) which provides builds for all major operating systems: Linux, Microsoft Windows, and Apple MacOS.
![google_chrome_logo_with_wordmark_(2015).svg.png](/google_chrome_logo_with_wordmark_(2015).svg.png =200x)
The CNF technical team recommends the following baseline computer hardware components to ensure a decent browser experience:
* **16 GB RAM minimum** (Current version of Microsoft Windows is v. 11 which is outrageously RAM hungry and therefore does not work smoothly with only 8 GB ram)
* Monitor resolution: 1600x900 (16:9) or 1680x1050 (16:10)
* Many users prefer a 2 monitor setup: 1 monitor for CNF and 1 monitor for email, other internet tabs, code set, etc.
* 250GB Hard drive
* Robust internet connection
* Connection to a printer and scanner

## FieldiNspect: Mobile device requirements
The CodeNforce handheld inspection software tool (i.e. "app") is released for the following mobile device platforms: 

* Android OS via Managed Play Private App
* iOS via the Apple App store

### Required features
* Camera (2MP+ resolution, which basically all modern smartphones have)
* GPS location services
* Internet connectivity

### Mobile device recommendations
**SMARTPHONES:** Users have satisfactorily used the handheld software on regular smartphones such as Apple's much-loved iPhone, Samsung Galaxy series smartphones, and others. 

**TABLETS:** Users who wish to use a larger-screened tablet for offline inspections are encouraged to purchase an **iPad generation 9+** with "Wi-Fi + Cellular". Users will find having 4G or 5G network access directly from the iPad to be very useful. **Wi-Fi–only iPads are not acceptable.** 

The Wi-Fi + Cellular line of iPads also has built-in GPS/GNSS which will be handy for future versions of the inspection tool which will geo-tag photos.

### About the handheld software
The software--currently in beta testing--allows users to conduct field inspections (take photos, mark violations, enter findings) while out of network service. Once network connectivity is restored, user data is automatically synced back to the server for final processing via the browser interface

### No Microsoft Surface support
The handheld software does NOT run natively as an app on MS windows and therefore will not run natively on MS Surface tablets. Users could still, of course, access CNF via a browser on a Microsoft Surface, but offline inspections would not be possible via the browser.
