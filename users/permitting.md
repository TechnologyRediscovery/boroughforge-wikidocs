# Permitting tools
All permitting occurs in objects called Permit Files. A permit file is a container for inspections, events, certificates, and random documents & images. 

## Permit file pages
[Creating a permit file](/users/permitting/creating-permit-files)
[Configuring canned comments on certificates](/users/permitting/managing-canned-cert-text)

## About permit files
Permit files always live inside a property unit. By default, properties are created with a unit called PRIMARY. Single family homes likely do not need any additional unit configruation and permit files can be created directly inside the PRIMARY unit. 

Multi-unit parcels should have their unit list configured to reflect the divisions actually present on the parcel. PRIMARY unit should either be deactivated or changed to become the common areas, or exterior of the property.
