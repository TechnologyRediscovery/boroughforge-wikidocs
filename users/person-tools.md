# Persons in CodeNForce
Persons in CodeNForce is the only search that crosses municipality lines, allowing municiplaites to share important data to leverage PA Act 90 if need be. It is important to note that when seeing the work "persons' in reference to CodeNForce, its refers to individual people or buinesses that are assiocated with properties. For example, a property can have linking with both the owner John Smith and the management company he employs, Rental Management LLC. 

No matter the type of person, the person profile has spots to enter in the same data points like the peron's mailing address, phone number, email address, and assiocation to the session property. Completing person profiles as the data comes in as well as updating roles helps ensure that everyone in a municipal office share data as well as others using the system in other municipalites. Most steps in CodeNForce will prompt the user to link a person to the action; whether a permit file, Code Enforcement Case, or citation, linking persons ensures that their data is tracked in the system. 

It is very important that before adding a new person in CodeNForce that users search for an existing link. This first step allows the data tracking to function as it should instead of muntiple persons being added to the system.
