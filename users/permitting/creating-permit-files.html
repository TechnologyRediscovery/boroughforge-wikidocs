How to Create a Permit File

Create a Permit file

Made by Natalie Merola with Scribe

creating a new permit file with optional steps to generate a report, certificate or reinspection



1. Navigate to the dashboard

2. search for the property

3. Once one property page -Click "Add new permit file"

4. Click "Choose a unit on this property" ( If there isn't a unit choose primary) or ad d a new

5. Click "select a permit file type..."- types are set up during onboarding, they are usually rental or property transfer. The permit file type holds the Certificate types

6. add origination category ( also set up with admin) such as application received. (What was the action that prompted the permit)this will be marked as an event

7. Click "Create new permit file and go to profile"

8. The permit file holds, the inspection, certificate and events etc. The origination event is viewed under events

9. Click "New inspection" to start an inspeciton

10. Click "edit checklist" - checklists can be set up with the admin or see checklist instructions

11. Click the checklist you are using

12. Click "Update inspection to use selected checklist"

13. Enter any inspection related data. Click disptatch to mobile if sending to the app.

14. Click "Create new inspection"

15. Click "Start inspection!"

16. Click here to select the spaces set up on the checklist.

17. Click "Inspect selected space"

18. Click "Table view & batch actions" ( at this time we do not recommend using tree view)

19. enter the status of the violation

20. Click "choose a batch status" if batching all, hint use batch status first then change any if needed

21. to add finding type in box, canned findings can be set up and stored.

22. Click "Save and continue to photos"

23. Click "Done! Back to inspection profile"

24. Click "Add Photos"

25. Click "Choose"

26. Click "Upload"

27. Click here.

28. Click "Inspect! edit ordinances pass/violphotos and findingsremove space"

29. Click "photos and findings"

30. Click "add/remove photos"

31. Click "attach to checklist item"

32. Click "Done! Return to space summary"

33. Click "Done! Back to inspection profile"

34. Click "finalize inspection"

35. Click "choose a determination"

36. Click "Violations identified (Fail)"

37. Click "Finalize inspection ID 4706"

38. if you would like a follow up event on the calendar please add in selected days. Click finalize and lock

Alert: Alert! - optional - a certificate can be created after the inspection is complete

39. Click "Create new certificate w/steps"

Alert: Alert! - optional an inspection report can be created

40. Click "Build Inspection report"

41. Click "Generate Inspection Report in new tab" - the report opens from a new screen - to print use ctrl +p

Alert: Alert! -optional an inspection report can be created after an inspection

42. Click "reinspect"

43. Click here to dispatch

44. Click "Generate follow-up inspection"

45. Click "Dispatched to mobile application." - the reinspect has been dispatched.
