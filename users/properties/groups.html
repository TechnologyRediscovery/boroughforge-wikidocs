Property Groups

1. Starting on the dashboard, click "property search & groups".

Tip: Always search for group before creating a new group to prevent duplicates. This helps the system flag for PA Act 90 and prevents information from being missed.

2. Click "New property group".

3. Type the name of the property group then click "Save new property group and close".

4. Navigate back to the property landing page and look to "Property Groups".

Tip: Start here if the group is already created and you need to add a property to the existing group.

5. Click "add to group..."

6. Click the drop down menu to see all the groups in the municipality.

7. Find the group you want to add the property to.

8. Click "Add 456 COGLAND TEST to selected group"
