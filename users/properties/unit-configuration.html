Property Unit Configuration

Made by Natalie Merola with Scribe



1. Start on the property landing page.

2. To the right of the screen, you will see the "Units, Permit Files, and Inspections box. Here you will see each unit associated with a property and the attached permit files.

3. Clicking "view/edit unit"...

4. will be able to see the unit information, permit files, mailing addresses, and person links.

5. To add a new unit, click "Add new unit"

6. For a single family home, "Primary" as the unit name is fine.

7. Or you can add unit numbers or names. This works for unit numbers in an apartment building.

8. It could also be another name, like "Basement" if there is another rental unit on the property.

Tip: Tip! The unit name/number will be visible on the printable permit file.

9. Here you can add mailing addresses for separate units.

10. Clicking "Persons" you can add tenants to the units.

Tip: Tip! Each unit has a permit file/inspection attached to it. Thus, even a single family home as a unit. However, this function helps make sure that individual rental units have separate permit files.
