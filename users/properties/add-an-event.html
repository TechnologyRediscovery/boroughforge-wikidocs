Add an Event to a Property

Made by Natalie Merola with Scribe



1. Start on the property landing page.

2. Scroll down to the "Events attached to parcels" box

3. Click "Add Event"

4. Choose the event type that best suits the event you're recording. This is just a general ty[e of event - you'll choose a more specific event category in the next few steps.

5. I chose "communication". Selecting an event type will filter out the event categories. making the next sept a little quicker.

6. Navigate to "Pick an event category"

7. Because I chose "Communication" as my event type, the system is showing me event categories that are communication categories. I chose phone call for creating this example.

8. Here you can see or change the "category details".

9. Click here to enter a description of the communication.

10. After entering a description of the event (what was discussed in a phone call/in-person conversation, etc) click "Create new event".

Tip: Tip! Adding these to a property can help you recall details about a property or property owner if ever asked about then in the future. This could be in preparation for court or for internal review purposes.
