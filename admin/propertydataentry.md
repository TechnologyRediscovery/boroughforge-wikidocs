# Property file data entry checklist
Prepared by Eric & Kim on 7-JAN-2026 for use during property file data entry in Sharpsburg.

1. Load property and configure addresses:
* Ensure correct property address is correct
* If a range address, include addresses in this range individually with link role "piece of range" and also include the range address itself should be role "property address"
2. Add property to group called "Rental property"
3. Set logical property use type under parcel info
3. Check and enter property owner name and contact objects on owner
* If we have an LLC and we know the LLC owner's name, make a person-person link to show ownership. Start at the LLC person and click "manage person links"
* Connect LLC mailing address to the LLC person profile
4. Check and enter property manager information: Property managers should be their own person records in CNF and linked to the parent property itself
5. Configure property units. 
* If units have a different mailing address than the parent property, link the unit to its own mailing address. 
* Create a permit file of type "Rental permit file"
6. Search for and find or add tenant persons with contact info
7. Link tenants to their respective units
8. Digest any and all hand written notes written directly on the property file folder and verify that the information in those notes is captured in the data entered in the steps above. Add any new information. Check person connections suggested by these notes.
9. Create event on parent property of type: Custom, Category: "Property file data entry". Include in the event description any odd data encountered.
