# Handy tools and references


## VS code KB shortcuts

Here are the essential VS Code keyboard shortcuts for Linux:

## Copilot Interaction
- **Ctrl+I** - Open Copilot inline chat (edit mode)
- **Ctrl+Shift+I** - Open Copilot chat panel
- **Tab** - Accept Copilot suggestion
- **Esc** - Dismiss Copilot suggestion
- **Alt+]** - Next Copilot suggestion
- **Alt+[** - Previous Copilot suggestion
- **Ctrl+Enter** - Open Copilot suggestions in separate panel

## File Navigation
- **Ctrl+P** - Quick Open (find files by name)
- **Ctrl+Shift+P** - Command Palette
- **Ctrl+Tab** - Cycle through open files
- **Ctrl+Shift+Tab** - Cycle backwards through open files
- **Alt+Left** - Go back in navigation history
- **Alt+Right** - Go forward in navigation history
- **Ctrl+Shift+E** - Focus on Explorer sidebar
- **Ctrl+Shift+F** - Focus on Search sidebar
- **Ctrl+B** - Toggle sidebar visibility

## Code Navigation
- **Ctrl+G** - Go to line number
- **F12** - Go to definition
- **Ctrl+F12** - Go to implementation
- **Shift+F12** - Show references
- **Alt+F12** - Peek definition
- **Ctrl+Shift+O** - Go to symbol in file
- **Ctrl+T** - Go to symbol in workspace

## Editing
- **Ctrl+/** - Toggle line comment
- **Ctrl+Shift+/** - Toggle block comment
- **Ctrl+D** - Select next occurrence of word
- **Ctrl+Shift+L** - Select all occurrences of word
- **Alt+Up/Down** - Move line up/down
- **Ctrl+Shift+K** - Delete line
- **Ctrl+Space** - Trigger suggestion/autocomplete
- **Ctrl+Shift+Space** - Trigger parameter hints

## Terminal
- **Ctrl+`** - Toggle terminal panel
- **Ctrl+Shift+`** - Create new terminal
- **Ctrl+Shift+5** - Split terminal

## Search/Replace
- **Ctrl+F** - Find in file
- **Ctrl+H** - Replace in file
- **Ctrl+Shift+F** - Find in workspace
- **Ctrl+Shift+H** - Replace in workspace
- **F3** - Find next
- **Shift+F3** - Find previous

## Multi-cursor
- **Ctrl+Alt+Up/Down** - Add cursor above/below
- **Alt+Click** - Add cursor at click position
- **Ctrl+U** - Undo last cursor operation

## Window Management
- **Ctrl+\** - Split editor
- **Ctrl+W** - Close editor
- **Ctrl+K Ctrl+W** - Close all editors
- **Ctrl+1/2/3** - Focus on editor group 1/2/3

## Pro Tips for Copilot
- Use **Ctrl+I** for quick inline edits without leaving your code
- Type `#file:filename.java` in Copilot chat to reference specific files
- Use `/fix` `/explain` `/tests` slash commands in Copilot chat
- **Ctrl+K Ctrl+I** - Show hover information (helpful for understanding code before asking Copilot)
